1.- python3 --version
    pip --version

2.- Instalar entorno virtual
    python3 -m venv env

3.- Activar entorno virtual
    win 
        env/Script/activate
    linux o mac 
        source env/bin/activate

4.- descargar las librerias desde nuestro txt
    pip install -r requirements.txt

5.- para ver todas las librerias 
    pip freeze
